# TaskManager Web App

Welcome to the TaskManager repository! This project serves as a showcase of my skills in both frontend and backend development. The web app is designed to efficiently keep track of tasks, demonstrating proficiency in a range of technologies.

## Features

- **Task Management**: Easily add, update, and delete tasks to stay organized.
- **Backend Development**: Utilizes Node.js, Express, and PostgreSQL to handle server-side operations.
- **Database Integration**: SQL and Sequelize enable seamless data storage and retrieval.
- **Frontend Design**: Responsive and user-friendly interface crafted with JavaScript.
- **External API Interaction**: AXIOS is employed for efficient communication with external APIs.
- **Cloud Deployment**: Hosted on Amazon EC2 for reliable and scalable deployment.

## Technologies Used

- **Frontend**:
  - JavaScript
  - HTML
  - CSS

- **Backend**:
  - Node.js
  - Express

- **Database**:
  - PostgreSQL
  - Sequelize (ORM for PostgreSQL)

- **External Communication**:
  - AXIOS

- **Cloud Deployment**:
  - Amazon EC2
